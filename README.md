# Excel Sales Dashboard Analytics
## Project Overview
This project presents an interactive Sales Dashboard developed in Microsoft Excel using Pivot Tables, Pivot Charts, and Slicers.
The dashboard helps analyze sales performance across multiple cities and sales executives while tracking target achievement and sales gaps.
---
## Dashboard Features
- Interactive City Slicers
- Total Sales Analysis
- Target Hit Percentage Tracking
- Away From Target Percentage Monitoring
- Pivot Tables and Pivot Charts
- Executive-wise Performance Analysis
---
## KPIs Included
1. Total Sales
2. Sales Target
3. Target Hit %
4. Away From Target %
5. Executive-wise Performance
---
## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
---
## Dashboard Components
### Dashboard 1
Bar Chart showing Total Sales by Sales Executive.
### Dashboard 2
Executive-wise Sales Summary Table.
### Dashboard 3
Target Achievement Percentage Analysis.
### Dashboard 4
Away From Target Percentage Trend Analysis.
---
## Business Insights
- Identify top-performing sales executives.
- Compare city-wise sales performance.
- Track achievement against targets.
- Detect executives requiring performance improvement.
